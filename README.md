# Library Management System Database Testing Project

## Overview
This project demonstrates end-to-end database testing for a Library Management System based on business requirements and BRD analysis.

The objective was to validate database design, data integrity, system workflows, and business rules through structured testing activities.

---

## Scope of Work

- Performed requirement analysis and identified requirement gaps.
- Designed database schema and Entity Relationship Diagram (ERD).
- Created SQL scripts for database creation and validation.
- Executed database test scenarios.
- Designed and executed functional test cases.
- Logged and tracked defects.
- Prepared UAT scenarios.
- Delivered test closure report.
